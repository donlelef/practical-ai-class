# GenAI
This folder contains notebooks to explore Generative AI concepts, including interacting with Large Language Models, embeddings, RAG, and agents.

## Getting Started
You will not need to install anything on your local machine. Just click the Colab links below to open each notebook in [Google Colab](https://colab.research.google.com/).
